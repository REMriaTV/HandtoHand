# Hand to Hand

短編小説「道程」の制作ログを公開するプレーンなサイトです。現状はスプレッドシートから転記したメモをまとめる場として使い、後々 CMS 化やインタラクティブな編集 UI を載せる想定です。

## 現状メモ
- **プロット管理**: `content/plot-outline.md` をベースに、章・節の更新があるたび手作業で貼り付け。
- **キャラクター**: `content/characters.md` に一次情報をまとめ、必要な部分のみここへ抜粋。
- **原稿メモ**: `content/manuscript.md` を参考に調整版を公開。

## ストーリー概要
### ログライン
女子と手を繋いだことのない男子中学生が、とある事件に巻き込まれるも、女子に手を差し伸べられて感動する話。

### テーマ
好きな娘としか手を繋ぎたくない主人公に試練と葛藤が訪れ、壁を破って小さな成長をする「思ってたのとちがうけど、悪くないかも！」という心の変化。

### 構想メモ
「好きな子と一緒になったら何がしたい？」と聞かれた瀬田が「ただ手がつなぎたい」と答えることから始まる、女子と手を繋いだことのない歴＝年齢の男子中学生の間抜けでまっすぐな青春譚。

## プロット概要
- **備考**: お題（手 / ハンカチ / 12時 / ちくわ / 嘘）は全編に絡ませ、章題としても機能させる案を検討中。
- **第1幕「手」/ 教室**: 瀬田が暴走発言で川原さんをドン引きさせ、クラスから孤立。
- **第2幕「ハンカチ」/ 渡り廊下**: 授業に戻らない川原を探しに出た瀬田が彼女のハンカチと遭遇。BGM: [Spotify](https://open.spotify.com/intl-ja/track/206TnJYAbYb6uFvFoPll0V?si=2c757dac279e4463)。
- **第3幕「12時」/ 体育館**: ハンカチを拾うか葛藤→意を決して体育館へ。体育教師に見つかりそうになり、床下ハッチへ逃げ込む。12時のチャイムで昼休み突入。BGM: Miliary Intelligence。
- **第4幕「ちくわ」/ 体育館地下倉庫**: ちくわ状と呼ばれるマットに挟まる展開を構築中。BGM: [zappaguri](https://youtu.be/DHVijvx-hG4?si=dRoSyJ3ySymGiyCY)。
- **終章「嘘」/ 体育館地下倉庫ほか**: 嘘をテーマにした締めの仕掛けと舞台を検討。BGM: What's the Internet。

## お題メモ
### 手
- 手を繋ぐこと自体が相手の情報を感じ取る神聖な行為。瀬田は手のコンプレックスや潔癖傾向をどう扱うか検討中。
- 横野に手を差し伸べられる終盤で、手繋ぎ童貞を誰に捧げるかという葛藤を深く描きたい。

### ハンカチ
- ハンカチは手の汗を拭う道具。川原は潔癖で基本持たず、横野が贈ったシルクのハンカチをお守りとして所持している設定案あり。
- この設定により、序盤で瀬田がハンカチに触れることの意味付けを再検討中。

### ちくわ
- バレー部員が「ちくわ」と呼ぶ体操マットを隠れ場所に使うアイデア。中抜き円柱構造やワープ装置的比喩、竹輪蒲鉾の由来などをヒントに演出を練る。

### 12時
- 正午チャイムが状況を悪化させるトリガー。学校生活の時間制約を活かし、昼休み直後の人流変化を描写。

### 嘘
- 誰がどのタイミングで嘘をつくか未確定。終章でテーマとして響かせるためにシーン案を募集中。

## キャラクター概要
- **瀬田亮二**: 異性と手を繋いだことのない歴＝年齢の男子。潔癖描写をどこまで採用するか検討しつつ、手へのこだわりで物語を牽引。
- **川原海美**: 才色兼備で医者志望。普段は授業を欠席しないが、瀬田の発言ショックで体育館へ。潔癖傾向やマイルールが課題となる。
- **横野志保（リンゴ）**: バレー部のムードメーカー。ノリで瀬田をからかいつつ、終盤で手を差し伸べる役割を担う。
- **クボセン / ザキヤマ / 盛岡 / バレーボール部員**: 学校生活のリアリティと緊張感を与える大人・同級生たち。

# キャラクター

主要人物と役割について、紹介します。

## 瀬田亮二
- 異性と手を繋いだことのない歴＝年齢の男子中学生。
- 手に対する潔癖と強いこだわりを持ち、「好きな娘としか手を繋がない」という信念が物語全体の葛藤を生む。
- 失言でクラスから浮き、体育館事件をきっかけに価値観を揺さぶられる。

## 川原海美（えみみ）
- 瀬田の意中の相手。才色兼備で東大志望、医者になる夢を持つ。
- 普段は遅刻・欠席をしないが、瀬田の暴走発言にショックを受け体育館へ。
- 潔癖傾向やマイルールがあり、他人と触れ合うこと自体が課題になっている。

## 横野志保（リンゴ）
- バレー部のムードメーカーで男子とも対等にふざけ合うタイプ。
- 瀬田の失言をツッコむ存在であり、終盤では実際に手を差し伸べるポジション。
- 川原の親友で、白いシルクのハンカチをプレゼントした張本人。

## 盛岡先生
- 体育教師でバレー部顧問。体育館の規律に厳しく、倉庫を地下にした改修に不満を持っている。
- 体力測定で体育館を使い、瀬田を窮地に追い込む存在。

## クボセン / ザキヤマ
- クボセン: 歴史担当。体調不良で欠席し、授業の穴を開ける。
- ザキヤマ: 公民担当で学年主任。自習監督を務め、瀬田と軽い会話を交わす。

## バレーボール部員
- 横野のチームメイト。倉庫での「ちくわ」シーンの相手役。
- 用具を探しに地下倉庫へ来て、瀬田のピンチを加速させる。

# 原稿メモ

`content/manuscript.md` にまとめている本文メモのサマリページです。章ごとの状態と読みどころ、気になる課題をざっと確認できます。

## 進行状況
- **（一）〜（七）**: 教室での失言から体育館地下倉庫までの出来事をラフに執筆済み。
- **（八）**: バレー部員との会話中心の書きかけ。以降は未着手。
- **今後**: 終章「嘘」の仕掛けと横野が手を差し伸べるクライマックスを執筆予定。

## 章ごとのハイライト
### （一）手
- クラスメイトとの雑談で「手を繋ぐだけでいい」と答えた瀬田が総ツッコミを受け、意中の川原海美に失言が届いてしまう。

### （ニ）ハンカチ
- 授業を抜けた瀬田が廊下で川原のハンカチを発見。間接手つなぎへの恐れと背徳感で10分間立ち尽くす。

### （三）体育館
- 新装された体育館の描写と、瀬田が川原を探し回る様子。体育教師に見つからぬようステージへ移動。

### （四）ステージ上
- 体育館で体力測定が始まり、盛岡の声に怯えつつ逃げ場を探す瀬田。舞台裏のハッチを偶然発見する。

### （五）反復横跳びの喧騒
- 盛岡の号令、反復横跳びの描写。瀬田は舞台裏で息を潜めながら脱出タイミングを伺う。

### （六）地下通路
- ハッチの下に滑り込み、真っ暗な通路で息を潜める。冷気と尿意に耐える描写が中心。

### （七）ちくわマット
- 倉庫に駆け込むも「ちくわ」状のマットに挟まれてしまい、横野の足音が近づく。ギャグと緊張が同居するシーン。

### （八・書きかけ）
- 倉庫に来た横野と友人の会話をメモレベルで記載。今後、マット越しのやり取りや嘘の要素を膨らませる予定。

## メモ・課題
- 横野が倉庫に来る理由（用具回収・昼練）を本文で自然に説明する。
- マットの高さや構造など物理描写を一貫させる。
- 川原の動機と「嘘」のテーマを終章で回収できるよう伏線を整備。

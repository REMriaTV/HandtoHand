# （八・書きかけ）

<div class="manuscript-scroll">
<div class="manuscript-body" markdown="1">

「ほんっとこの倉庫使いにくいよね、まじなんでわざわざ地下に埋めたんだよ。」
「うん。改装して使いにくくなるの謎すぎる...」
「あはは...バレーの用具出すだけで準備運動ってブラック企業ですかマジで！」
「それな！リフトあるのに、滅多なことじゃ使わせてくれないし、意味わかんないよね」
横野の一緒にいるもう一人は瀬田の知らない声だが、他のクラスの生徒だろう。二人はマットレスのすき間に同級生が挟まっているなどつゆ知らず、言い慣れた愚痴を吐きながら瀬田のすぐそばを通っていった。
「盛岡センセイも物忘れ激しいくせに、ウチらが物なくすとすごい怒るよね。」
「うん...てかシホリン、本当に倉庫に置き忘れたの？」
「うーん、わからない。昨日いっぱい走った後だったから。」
バレーボール部は中学を代表する強豪部活の一つで、全国大会の常連だ。顧問の盛岡は昔ながらの走り込みスタイルで部員を震え上がらせるが、横野は男子顔負けの体力で女子のエースに君臨している。
「まじ？やば。しかもよりによって盛岡センセーの愛用タイマーなんでしょ。」
「愛用っていう割にはボロボロだけどね。」
「それは言っちゃいけないやつ！」
マットに埋もれた瀬田には2人の様子を直接目にすることはできない。
「やっぱないかー。まあ、盛岡センセーのことだから、自分のトレパンのポケットから出てくるんじゃない？」
「まあたしかに、それくらいのことはあるかもね。」
「だよね！じゃあ上いこっか。」
大したやる気のない探し物はすぐに終わり、2人は部屋を後にした。腹の空く昼休みに真っ先に倉庫へ来たのには、2人が早弁をすでに済ませ昼練を控えているからだった。

# （七）

<div class="manuscript-body">

この状況で見つかれば、妄想好きの変態どころか倉庫に侵入したコソ泥認定されるのは間違いない。扉を開けて入ってくる者が誰であっても自分は100％マズい状況に置かれるだろう。脳内でそんなことを考える暇さえなく、再び訪れたピンチに瀬田の身体中を電気が駆け抜けた。鍵を開けようと音のするドアから瞬時に飛び去ると、舞台へ上がるためのハッチの下へ暗闇を急いだ。倉庫に入ってくるのが誰であれ、舞台の上に出てしまえばこっちものだ。だが闇は依然として濃い。猫の目を持たない瀬田はガタッという大きな物音を立てて思いきり何かにつまずいた。体勢を崩した瞬間、宙に浮かんだ瀬田は次に起こる衝撃に備えて全身に力を込めた。ドスン。鈍い衝撃音は幸いにも瀬田の最悪のシナリオから外れていた。その感触は金属ラックの鋭利なフレームでもコンクリートの冷たい壁でもない、なんとも馴染みのあるものだった。記憶の内を懸命に脳内検索して弾き出された答えに辿り着くと、瀬田は渾身の力で身体を押し上げた。
 
「やっと開いたよ...新しい鍵って馴染むまで開きにくいんだよね〜。」
背後で声がしたことに、無我夢中でよじ登りそこにあるすき間に身体を押し込もうとしていた瀬田には気づく余裕がなかった。穴に迷わず足をまず入れ、次いで胴を押し込んだ。スポンッと音がするように体は一気に構造物の中心へ吸い込まれていき、自然と顎の上がった格好になった瀬田の目に天井が飛び込んできた。グレーの下地材に無数の亀裂が入っているように見える天井の化粧は、防音や耐震などに工夫された構造である。瀬田が隠れるのを見計らったように、部屋の電気がついた。
「てか、さっき開ける前、なんか物音しなかった？」
「え？なに！？・・・怖ぁ！ヤダヤダそういうの。」
「いや・・・気のせいかな。前はなんか出たらしいけど。」
「だからやめてってば！私そういうのダメだからねっ！」
部屋に入ってきた声の主のうち、1人はよく知った声だった。横野だ。手を繋いであげると上から目線で挑発し、瀬田の変態発言を引き出した張本人である。
 
咄嗟にしては完璧だった。芸術点は極めて高い。暗闇の中で躓きバランスを崩した瀬田が前からのめり込むようにして激突したのは分厚い体操マットだった。使う時は床に敷くが、収納時にはくるくると巻いて縦置きすれば自立する。スチールラックや跳び箱などの硬い素材がひしめく中で、１箇所にまとめ置かれた数ロールの体操マットにぶつかったのは瀬田がまだ神様に見捨てられていなかったことの証拠だ。そしてここぞとばかりに瀬田の脳内コンピューターが過去最高のひらめきを見せて芸術点満点の滑り込みを見せた。...あるいは、歴史の答案用紙に落書きした石の穴に挟まれた西洋人のイラストや数十分前にも通ってきた舞台床の地下通路の記憶がまだ頭にあっただけかもしれないが。

</div>

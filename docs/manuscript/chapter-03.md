# （三）

<div class="manuscript-body">

なめらかな光沢を放つシルクの生地に雪のような小花の刺繍が可愛らしくあしらわれ、端についたレースにまできちんとアイロン掛けされていて一切の皺や捻れがない。いつも使ってるはずなのにまるで新品のように綺麗な風合いで、これぞまさしく川原さんがいつも机の上に置いている愛用のハンカチだった。でも、それがどうしてここに？答え合わせを求めて視線を廊下の奥へ向けると、体育館の扉が中途半端に開けられていた。
 
川原さんが授業を放って体育館へ逃げ込むほど緊急事態に追い込まれた理由が自分のくだらない発言のせいだと瀬田は思いたくなかったが、扉の向こうに川原さんがいるのはほぼ確実だった。普通の男子ならハンカチを渡しに行く体をよそおいながら「さっきのはギャグだから気にしないでね」「あれは横野に向けて言ったんだ」などと弁解しに行くところだが、瀬田は全くそれどころではない別の深刻な問題に出くわしていた。それは、女子と手を繋いだことのない自分がハンカチを拾えば、間接キスならぬ''間接手つなぎ''に当てはまるのではないだろうか、という恐れだった。相手はあの川原さんで、彼女の分身とも言える聖なる手巾に本人のいないところで勝手に触れるなど、倫理的に許されるはずがなかった。生地には彼女の手汗に皮脂、吐息から付着した唾液の成分などが多分に含まれており、触れた瞬間にそういった超個人的・生データを瀬田は一方的に受け取れてしまうのだ。たかがハンカチ...、されどハンカチ。単純な動作の裏にある特別な背徳感が瀬田を異様なほどの緊張感と高揚感に包み込み、王室警備兵のように直立不動の姿勢を取らせた。
 
約10分、渡り廊下の真ん中で地面に落ちている白いハンカチを前に立ち尽くしていた瀬田は、３時限目の終わりを告げるチャイムが鳴り響くと体に電気が走ったかのようにビクッとした。遠目に運動着に着替えた生徒たちが体育館の方へやってくるのが見えると、我にかえった瀬田は教科書通りのコソ泥の動きで地面からハンカチを掴み取った。その際、ワイシャツの袖を伸ばしてハンカチには直接触れぬことのないよう慎重に配慮しながら行ったのは言うまでもない。''手つなぎ童貞''の誇りにかけて、自分の初めてを卒業する際には事前に間接手繋ぎを経るなどという邪道なことは決してせず、正々堂々と恋人になってから手を繋ぐ行程を行くのだと、瀬田は心に決めていたのだ。

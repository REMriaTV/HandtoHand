# （四）

<div class="manuscript-body">

瀬田の通う中学校は市内では歴史がある方で校舎も古ければ在籍する教員もベテラン揃いだ。伝統的に運動部活が盛んで全国大会に出場しているクラブも一つや二つではない。日頃のトレーニングの要である体育館は2年かけて春に改築したばかりで、創建当時の埃臭い教室棟から渡り廊下を通って体育館へ入ると、床や壁から新しい建材の匂いが漂ってくる。以前にも増して気合いの入る体育教師陣は晴れていてグラウンドが使える日でも体育館を使用し、バスケットボールやバレーボールなどの球技以外にリトミックやダンスなど室内運動のレパートリーを増やすようになった。
 
４時限目は生物のはずだったが川原さんは一体どこにいるのだろうか。張り替えて間もない色の浅いフロアの上に川原さんの姿はなかった。
（医者を目指す上で重要な理科系科目を休むなど川原さんにはしてほしくない。俺のことは嫌いでも授業にはちゃんと戻ってきてほしい！）
真正の川原推しだからこそ自らをおいて川原のことを心配する瀬田だったが、心のどこかでは変態男子としか見られていない自分がよもや愛用するハンカチを持っていると川原さんに知れた時には、余計に自分のことが嫌いにならないかという不安に駆られていた。
 
休み時間が残り５分になっても廊下で見かけた体育着の生徒らは誰1人として体育館に入ってこなかった。
（そういえば、今週から体力測定をはじめると体育教師の盛岡が言ってたっけ。グラウンドで測定しなければならない種目もあるから、さすがに今日は室内じゃないのかな。）
なんとなく瀬田はそう決めつけると、体育館の奥にある一段上がったステージに上がり舞台袖のを探しに行った。
（川原さんを見つけたら、''ハンカチを渡り廊下で拾った''とだけ言ってすぐに引き返そう。４時限目がすでに始まっていて、大事な生物の授業に姿を見せない川原さんのことを心配して来たのだといえば、川原さんも少しは俺の印象を変えてくれるかなぁ。）
緞帳の前を行ったり来たりしながら瀬田が楽天的な妄想に花を咲かせていると、突如ガヤガヤと大勢の声が体育館の入り口の方から聞こえて来て体操着の生徒たちがなだれ込んできた。
 
「......じゃあ、残り時間は室内種目を測定しまァす。今から反復横跳びを測るので、各自ストレッチなどもう一度入念にしておくように！５分後に第一班からスタートしまァす！」
筋金入りのマッチョ教師である盛岡の野太い声が地響きのように体育館を震わすと、遅れて入って来た生徒たちも怠そうに予備運動を始め出した。まさか前半後半で分けてまで体育館を使うとは思ってなかった瀬田は、慌てて身を隠すところを探した。規律に厳しい盛岡にステージ裏から出てくるところを見られたら一体何を言われるかわからない。

</div>

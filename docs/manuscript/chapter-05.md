# （五）

「ではァ！一班はこのラインに並んで、3本の線の中心に立ってください！」
盛岡の指示が体育館に響くと、最初の10名程度が体育館の中ほどから入り口までの間に一列に並んだ。
「２班と３班の人は何回ラインを越えたかカウントしてあげてください。今から２０秒間、自分の一番速い動きで3本の線を右、左、右、左って行き来してください！」
年齢の割に機敏な動きで盛岡が反復横跳びの動作をやってみせると、室内履きがキュッキュッと摩擦音を鳴らした。
「では、はじめ！」一瞬の静寂ののち、今度は１０人ほどの生徒らが激しく床を踏み鳴らし始めた。ひとまず緞帳に隠れていた瀬田は、ステージからの脱出を図るべく今だとばかりにと舞台袖の方へ移動した。しかし無情にも２０秒というのは体育館脱出の糸口を見つけるには短すぎた。舞台裏には独立した出口がなく、皆がいるフロアを通り抜けずには脱出できないことを悟るので精一杯の時間だった。
 
「はいやめー！２班３班の人はペアの人の跨いだ回数を確認し合って、紙に書いてくださいね。おっと、紙を忘れた...。じゃあ、第二班は先生が紙を持ってきたら測定開始するから準備しておいて！」
盛岡は記録を書き込む用紙を取りにどこかへ向かって歩み出した。その足音は段々と舞台袖で息を潜めている瀬田の方に近づいてくるではないか。そういえば、舞台に上がる時にフロア側の舞台縁に計測用紙の入ったカゴを見た気がする。あれか。瀬田は近寄ってくる盛岡に気づかれないように舞台の真裏へ回り込んだ。真裏の通路は人が2人やっとすれ違えるほどの狭さで真っ暗だった。しばらくここにいよう。瀬田はそう思いじっとしていると、どしんという重そうな音とともに盛岡がステージへ上がる気配があった。なぜだ。舞台奥には何も用はないはず。しかし盛岡はどこへやったんだっけなぁと独り言を言いながら舞台袖を物色している。ベテランのくせして物の管理がいい加減なのかよ、と瀬田は心の中で突っ込んだ。
 
「新しくなったはいいけど...用具庫が同じフロアにないってのはいただけんな...。コートを多く取りたいからってなんで倉庫を地下にしたんだよ...！ほんと現場を何も分っちゃいねえ...これだから勉強しかやってない奴は駄目なんだよ。」
ぶつぶつと愚痴を吐きながら盛岡が袖から舞台裏に回り込んでいくと、瀬田は追われるように舞台袖に戻らざるを得なかった。このまま盛岡が再び舞台袖を通ってフロア側に戻ってきたら自分にはもう隠れる場所がない。追い詰められた瀬田が立ちすくすと、ふいに舞台の照明が点いた。盛岡が真っ暗な舞台裏を探すために点けたのだった。すると瀬田の目の前の床に野球のベースほどある四角い枠線が浮かび上がった。それはハッチタイプの扉で、床下の空間がある可能性を示していた。表側に収納されたフックを瀬田が指で引っ掛けて持ち上げると、意外にもハッチはすんなりと開き、中のひんやりした空気が浮き出てきた。もはや時間はない。手をついた床に伝わる振動で盛岡がすぐ背後まで迫っていることを知った瀬田は、急浮上した目の前のアイデアを試すしかなかった。

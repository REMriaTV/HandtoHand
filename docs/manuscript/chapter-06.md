# （六）

「チッ、なんでこんなところにあるんだよ。」
明るくなった舞台裏でようやく目当てのものを見つけた盛岡が頭上の床を踏み鳴らしてフロアへ戻っていくのを、狭い空間に身体をねじ込んだ瀬田はヒヤヒヤしながらやり過ごした。土壇場で瀬田が入り込んだのは、舞台で催し物をする際に小道具などを地下に収納するためと思われる通路だった。階下には階段が続いていて瀬田はその空間に身体を完全に隠すことができた。盛岡があと数メートルまで迫ってきた時、慌てて中からハッチを閉めると思わず音が鳴ってしまったが、幸いにして盛岡が気づくことはなかった。光を失った地下空間は不気味な静寂を放っていた。
 
実は、瀬田は授業をサボるのも学校内の施設に忍び込むのもこれが初めてだった。未経験のことに心臓はバクバクとアラームを鳴らし、見通しの悪い闇の中で声高に不安を叫んだ。
一体自分は何をしているんだろう...。
いまやすっかり当初の目的だった川原さん探しは二の次で、授業をサボって体育館に潜り込んでいるのをいかに盛岡に見つからずにいられるかに奔走していた。
階段を数段下がったところで動かずにいると、階下からせり上がる冷気が背中に汗をはり付かせて気持ちが悪かった。この空間の不気味さに瀬田は一刻も早く外に出たかったが、ハッチを閉めると防音性が意外にも高く扉の向こうの様子はまるで分からなかった。万一、盛岡や生徒たちが舞台の方を見ている前で床から首だけひょこっと出しているのを見つかったら、失笑ものだ。盛岡にこっぴどく怒られるのも嫌だったが、変態のイメージがさらに上書きされることを瀬田は恐れた。瀬田はなんとか我慢して昼休みになるまで待つことに決めた。
 
しばらくすると、すっかり体が冷えてしまったからか尿意を覚えた。ここが盛岡の言う倉庫ならば、階下の廊下に面しているはずだ。瀬田は足裏の感覚で段差を確かめつつ床に降り立つと、暗闇に手を伸ばして部屋の中を確かめた。触った感触で壁沿いに金属製のラック棚のようなものが置かれているのが分かった。棚にはボックスやら何かのトレーニング器具やらが隙間なく置かれていて、瀬田はここが倉庫であると確信した。ブラインド状態で出口を探そうと壁沿いに歩くも思っていたより広い部屋に瀬田が不安を抱き始めた時、お化け屋敷の仕掛けのようにけたたましくチャイムがなった。学校全体が昼休みに入ったことを示す12時のベルだった。
助かった！
トイレを我慢するあまり小刻みに足を動かしながら、あと５分もすれば生徒たちが引き上げて無事に自分も体育館を脱出できるのだと瀬田が安堵したのも束の間、勢いよくドアノブを回す音が聞こえた。廊下に面した倉庫のドアから誰かが入ってこようとしているのだった。

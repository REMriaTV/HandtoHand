# プロット詳細

お題「手 / ハンカチ / 12時 / ちくわ / 嘘」を軸に、短編「道程」の進行を章ごとに整理しています。音楽リンクは執筆時のイメージトラックです。

## 第1幕「手」
- **あらすじ**: 女子と手をつないだことのない瀬田亮二が、幼馴染の横野に煽られた結果の暴走発言が、意中の川原海美をドン引きさせる。
- **できごと**: 瀬田の手繋ぎ童貞がクラス全員にバラされる。変態発言を聞いた川原が教室を飛び出す。
- **メモ**: 「瀬田が女子と手をつなげるのか」が主題になる。
- **場所**: 3年3組の教室内。

## 第2幕「ハンカチ」
- **あらすじ**: 授業に戻らない川原を探しに廊下へ出た瀬田が、見覚えのある白いシルクのハンカチを見つける。
- **できごと**: 前期末試験で書いたケルトの穴空き巨石を思い出す。「間接手つなぎ」に葛藤しつつも、ハンカチを拾う。
- **BGM**: [Spotify リンク](https://open.spotify.com/intl-ja/track/206TnJYAbYb6uFvFoPll0V?si=2c757dac279e4463)。
- **メモ**: ハンカチに触れること＝間接的に手に触れる行為。瀬田にとってはパンツを拾うようなもの（笑）
- **場所**: 教室から体育館へ続く渡り廊下。

## 第3幕「12時」 / 体育館 & ステージ
- **シークエンス**:
  - ハンカチを袖越しに拾い、体育館内を探索。
  - 体育教師・盛岡に見つかりそうになり、舞台裏から床下のハッチへ逃げ込む。
  - 薄暗い地下空間で尿意と不安に襲われる中、12時のチャイムが昼休みの開始を告げる。
  - BGM: Military Intelligence。
- **ポイント**: 12時が状況悪化のトリガーになり、瀬田の孤立感を強調。

## 第4幕「ちくわ」 / 体育館地下倉庫
- **状況**:
  - 倉庫で「ちくわ」と呼ばれる体操マットのすき間にハメられる瀬田。
  - バレー部員（横野志保ほか）が用具を探しに来て、さらに逃げ場がなくなる。
  - BGM: [zappaguri](https://youtu.be/DHVijvx-hG4?si=dRoSyJ3ySymGiyCY)。
- **課題**: ちくわ比喩やマット構造の描写を詰め、瀬田の心象とリンクさせる。

## 終章「嘘」 / 体育館地下倉庫 ほか
- **未決事項**:
  - 誰がどの嘘をつくか、そしてその嘘が瀬田の成長にどう作用するかを検討中。
  - BGM: What's the Internet。
- **ゴール**: 横野に手を差し伸べられた瀬田が、こだわりや潔癖を乗り越えて「悪くないかも」と思える結末。
